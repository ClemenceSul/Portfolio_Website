Discover Windows

In older generations, computer illiteracy causes disparities, isolation or dependancy. Following insightful user research, I created this webapp as an easy interactive guide to give the elderly the necessary tools to communicate, understand and use their computers.

Discover Windows

I discovered seniors learnt about their computers best with visual simulations, mirroring displays they really encounter on their computers. Furthermore, I noticed seniors often don’t click on unknown buttons, wary of causing an unwanted reaction. Thus, I decided to recreate common windows with buttons activating when hovered upon. Guidelines were also displayed on each page to inform users on how to use it.

Programming

For this project, I taught myself HTML, CSS and Javascript, to obtain the functionalities user’s wanted.

User Collaboration

Performed several interviews, observations, and validation tests in order to pinpoint how to best serve the targeted audience.

UX / UI Design

Accessibility and intuitiveness of the webapp was designed for the elderly according to the insights gained from users.
